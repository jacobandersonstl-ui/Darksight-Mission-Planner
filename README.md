Darksight Aerospace Mission Planner

This is a project I decided to make because I was bored and wanted to give my KSP save some substance. it is a "Mission Planner" that allows you to edit the orbits of fake satellites launched in my KSP save, Darksight Aerospace.

Current Satellites:

Forge II
Forgiven II
More Coming Soon

Features:

Minimal Orbital editing for Forge II Satellite
Minimal Terminal/ASCII based UI
that's really it for now

How to Run:

Python Source:

Requirements;
Python 3.14 or newer, that's it!

Download Darksight_Aerospace_Mission_Planner.py
Open the file in VS Code (make sure you have the Python extension!)
Run it!

Or

Download Darksight_Aerospace_Mission_Planner.py
Open Powershell
Use the 'cd' command to change to the folder in which you downloaded the file too (Most commonly 'C:\Users\your_user\Downloads')
Run 'Python Darksight_Aerospace_Mission_Planner.py
